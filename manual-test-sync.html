Troubleshooting

If Login Doesn't Work:

  1. Open browser developer console (F12)
  2. Check for Firebase errors in the console
  3. Make sure the test user exists in Firebase Authentication
  4. Verify Firebase Realtime Database is in test mode

If Sync Button Doesn't Appear:

  1. Refresh the page (Ctrl+R or Cmd+R)
  2. Make sure you're logged in (check for user menu in top-right)
  3. Verify you're on the Games page (/games route)

If Sync Fails:

  1. Check browser console for errors
  2. Verify Firebase Realtime Database rules are set to test mode
  3. Check if games already exist (sync won't duplicate)
