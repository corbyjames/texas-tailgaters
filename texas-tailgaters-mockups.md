# Texas Tailgaters - Screen Mockups & UX Design

## Design System

### Color Palette
- **Primary**: UT Burnt Orange (#BF5700)
- **Secondary**: UT White (#FFFFFF)
- **Accent**: UT Dark Gray (#333F48)
- **Success**: Green (#10B981)
- **Warning**: Yellow (#F59E0B)
- **Error**: Red (#EF4444)
- **Background**: Light Gray (#F9FAFB)
- **Text**: Dark Gray (#1F2937)

### Typography
- **Headers**: Inter, Bold, 24px/20px/18px
- **Body**: Inter, Regular, 16px
- **Captions**: Inter, Medium, 14px
- **Buttons**: Inter, SemiBold, 16px

### Spacing
- **Container**: 16px padding
- **Elements**: 12px gap
- **Sections**: 24px margin

## Navigation & Interaction Patterns

### Bottom Navigation (Mobile)
```
┌─────────────────────────────────────┐
│                                     │
│ [📱] [🏈] [👥] [⚙️]                │
│ Home  Games  Group  Settings        │
└─────────────────────────────────────┘
```

### Quick Actions
- **Swipe left/right**: Navigate between games
- **Pull to refresh**: Update schedule data
- **Long press**: Quick actions menu
- **Double tap**: Mark item as favorite

### Loading States
- **Skeleton screens**: For schedule loading
- **Progressive disclosure**: For potluck items
- **Optimistic updates**: For RSVPs and signups

### Error States
- **Offline mode**: Cached data with sync indicators
- **Empty states**: Helpful prompts for first-time users
- **Error messages**: Clear, actionable feedback

---

## Responsive Breakpoints

### Mobile (320px - 768px)
- Single column layout
- Bottom navigation
- Touch-friendly buttons (44px minimum)
- Swipe gestures

### Tablet (768px - 1024px)
- Two-column layout where appropriate
- Side navigation option
- Larger touch targets

### Desktop (1024px+)
- Multi-column layouts
- Hover states
- Keyboard shortcuts
- Advanced admin features

## Accessibility Features

- **Color contrast**: WCAG AA compliant
- **Screen reader**: ARIA labels and semantic HTML
- **Keyboard navigation**: Full keyboard support
- **Focus indicators**: Clear focus states
- **Text scaling**: Support for 200% zoom
- **Motion reduction**: Respect user preferences
